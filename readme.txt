| DESCRIPTION |
+-------------+

With this BBcode you can hide some text with a spoiler.


+--------------+
| INSTALLATION |
+--------------+

1. Upload all files and folders to your ftp root;
2. Go to Admin -> System Administration -> BBCodes and 
3. just enable "spoiler2" BBCode, that's it!
